# YelpCamp
강의 내의 클론 코딩 프로젝트입니다. 미국 내의 캠프장과 리뷰를 등록, 수정, 삭제할 수 있습니다. 
[(링크)](https://guarded-wildwood-42249.herokuapp.com/)

### 주요 기능
1. 캠프장 등록 / 수정 / 삭제
2. 캠프장 게시물에 리뷰 등록 / 수정 / 삭제
3. 회원가입 / 로그인
